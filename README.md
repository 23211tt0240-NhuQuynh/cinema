# Mobile Feature Showcase

A collection of core independent modules and features developed to practice mobile application architecture and backend/logic integration.

## Key Modules & Features

### 1. Membership & Tier Management
Handles the core logic to track user loyalty points and automatically upgrade or downgrade membership tiers.

### 2. Movie & Showtime Scheduler
Builds the database structure and implements CRUD (Create, Read, Update, Delete) operations to manage movie catalogs and configure showtime schedules.

### 3. Two-Way Admin & User Support
Allows users to submit support requests, while administrators can view, manage, and respond to all incoming inquiries. Users can then seamlessly read the responses sent back by the admin.
